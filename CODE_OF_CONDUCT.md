# Code of Conduct

## Our Commitment

TradePilot OS aims to maintain a respectful, professional, and collaborative development environment.

## Expected Behavior

Participants should:

- Be respectful.
- Be constructive.
- Focus on technical discussions.
- Help improve the project.
- Respect differing viewpoints.

## Unacceptable Behavior

The following behaviors are unacceptable:

- Harassment
- Personal attacks
- Discrimination
- Offensive language
- Intentional disruption

## Enforcement

Project maintainers may remove contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to:

- Source code
- Documentation
- Discussions
- Pull requests
- Issue tracking

Thank you for helping build a professional project.
