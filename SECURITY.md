# Security Policy

## Supported Versions

During active development, only the latest development version is supported.

## Reporting a Vulnerability

If you discover a security issue:

1. Do not publish the vulnerability publicly.
2. Report the issue privately to the project owner.
3. Include:
   - Description
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Potential impact

## Security Principles

TradePilot OS aims to:

- Protect user data.
- Store sensitive information securely.
- Minimize attack surface.
- Follow secure coding practices.
- Regularly review dependencies.

## Future Enhancements

The following security features are planned:

- Encrypted credential storage
- Secure configuration management
- Audit logging
- Role-based access control
- Secure update verification
